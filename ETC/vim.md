# Vim cheatsheet

## Navigation

```
h	cursor left
j 	cursor down
l	cursor right
k	cursor up
```

```
ctrl + b	page up
ctrl + f 	page down
G			bottom of file
gg			top of file
#G			goto line #
#gg			goto line #
```

```
w	jump by start of words
e	jump to end of words
b	jump backward by words
^	first of line
$ 	end of line
```
